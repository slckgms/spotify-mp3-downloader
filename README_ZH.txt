关于本程序
----------
Spotify MP3 下载器可让您通过 Spotify 播放列表、专辑或歌曲链接
免费以高质量下载音乐。下载通过 YouTube Music 进行。

功能特点
--------
✓ 支持播放列表、专辑和单曲
✓ 格式支持：MP3、M4A、OGG、OPUS、AAC、FLAC、WAV、AIFF、ALAC
✓ 音质选项：128 / 192 / 256 / 320 kbps
✓ 自动添加元数据（标题、艺术家、专辑）
✓ 封面嵌入（MP3 和 M4A）
✓ 断网自动重试
✓ 支持超过 100 首歌曲的播放列表
✓ 支持 12 种语言

使用方法
--------
1. 在 Spotify 中打开所需的播放列表/专辑/歌曲
2. 分享 > 复制链接
3. 粘贴链接（Ctrl+V 或右键 > 粘贴）
4. 点击「获取」
5. 选择格式和音质
6. 点击「下载」
7. 完成后点击「打开文件夹」

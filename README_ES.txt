ACERCA DEL PROGRAMA
-------------------
Spotify MP3 Descargador le permite descargar música gratis en alta calidad
usando enlaces de playlist, álbum o canción de Spotify.
Las descargas se realizan a través de YouTube Music.

CARACTERÍSTICAS
---------------
✓ Soporte para playlists, álbumes y canciones individuales
✓ Formatos: MP3, M4A, OGG, OPUS, AAC, FLAC, WAV, AIFF, ALAC
✓ Calidad: 128 / 192 / 256 / 320 kbps
✓ Metadatos automáticos (título, artista, álbum)
✓ Inserción de portada (MP3 y M4A)
✓ Reintento automático al perder la conexión
✓ Playlists con más de 100 canciones compatibles
✓ Soporte para 12 idiomas

CÓMO USAR
----------
1. Abra la playlist/álbum/canción en Spotify
2. Compartir > Copiar enlace
3. Pegue el enlace (Ctrl+V o clic derecho > Pegar)
4. Haga clic en "Obtener"
5. Elija formato y calidad
6. Haga clic en "Descargar"
7. Al terminar: "Abrir carpeta"

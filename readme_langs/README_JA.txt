プログラムについて
------------------
Spotify MP3 ダウンローダーは、Spotifyのプレイリスト、アルバム、
または曲のリンクを使用して、無料で高品質の音楽をダウンロードできます。
ダウンロードはYouTube Musicを通じて行われます。

機能
----
✓ プレイリスト、アルバム、単曲のサポート
✓ 対応フォーマット: MP3、M4A、OGG、OPUS、AAC、FLAC、WAV、AIFF、ALAC
✓ 音質オプション: 128 / 192 / 256 / 320 kbps
✓ 自動メタデータ（タイトル、アーティスト、アルバム）
✓ カバーアート埋め込み（MP3 と M4A）
✓ 接続切断時の自動再試行
✓ 100曲以上のプレイリスト対応
✓ 12言語サポート

使い方
------
1. SpotifyでプレイリスB/アルバム/曲を開く
2. 共有 > リンクをコピー
3. リンクを貼り付け（Ctrl+V または右クリック > 貼り付け）
4.「取得」をクリック
5. フォーマットと音質を選択
6.「ダウンロード」をクリック
7. 完了後:「フォルダを開く」

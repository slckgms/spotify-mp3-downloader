À PROPOS
--------
Spotify MP3 Téléchargeur vous permet de télécharger de la musique gratuitement
en haute qualité via des liens Spotify (playlist, album ou titre).
Les téléchargements sont effectués via YouTube Music.

FONCTIONNALITÉS
---------------
✓ Support des playlists, albums et titres individuels
✓ Formats : MP3, M4A, OGG, OPUS, AAC, FLAC, WAV, AIFF, ALAC
✓ Qualité : 128 / 192 / 256 / 320 kbps
✓ Métadonnées automatiques (titre, artiste, album)
✓ Intégration de pochette (MP3 et M4A)
✓ Reconnexion automatique en cas de coupure
✓ Playlists de plus de 100 titres supportées
✓ Support de 12 langues

UTILISATION
-----------
1. Ouvrez la playlist/album/titre sur Spotify
2. Partager > Copier le lien
3. Collez le lien (Ctrl+V ou clic droit > Coller)
4. Cliquez sur "Récupérer"
5. Choisissez le format et la qualité
6. Cliquez sur "Télécharger"
7. Terminer : "Ouvrir le dossier"

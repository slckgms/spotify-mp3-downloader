SOBRE O PROGRAMA
----------------
O Spotify MP3 Downloader permite baixar música gratuitamente em alta qualidade
usando links de playlist, álbum ou música do Spotify.
Os downloads são feitos via YouTube Music.

RECURSOS
--------
✓ Suporte a playlists, álbuns e músicas individuais
✓ Formatos: MP3, M4A, OGG, OPUS, AAC, FLAC, WAV, AIFF, ALAC
✓ Qualidade: 128 / 192 / 256 / 320 kbps
✓ Metadados automáticos (título, artista, álbum)
✓ Incorporação de capa (MP3 e M4A)
✓ Nova tentativa automática ao perder conexão
✓ Playlists com mais de 100 músicas suportadas
✓ Suporte a 12 idiomas

COMO USAR
---------
1. Abra a playlist/álbum/música no Spotify
2. Compartilhar > Copiar link
3. Cole o link (Ctrl+V ou clique direito > Colar)
4. Clique em "Buscar"
5. Escolha formato e qualidade
6. Clique em "Baixar"
7. Ao concluir: "Abrir pasta"

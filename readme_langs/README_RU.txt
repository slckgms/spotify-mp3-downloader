О ПРОГРАММЕ
-----------
Spotify MP3 Загрузчик позволяет бесплатно скачивать музыку в высоком качестве
по ссылкам на плейлисты, альбомы или треки Spotify.
Загрузка выполняется через YouTube Music.

ВОЗМОЖНОСТИ
-----------
✓ Поддержка плейлистов, альбомов и отдельных треков
✓ Форматы: MP3, M4A, OGG, OPUS, AAC, FLAC, WAV, AIFF, ALAC
✓ Качество: 128 / 192 / 256 / 320 kbps
✓ Автоматические метаданные (название, исполнитель, альбом)
✓ Встраивание обложки (MP3 и M4A)
✓ Автоповтор при обрыве соединения
✓ Поддержка плейлистов с более 100 треками
✓ Поддержка 12 языков

КАК ИСПОЛЬЗОВАТЬ
----------------
1. Откройте нужный плейлист/альбом/трек в Spotify
2. Поделиться > Скопировать ссылку
3. Вставьте ссылку в программу (Ctrl+V или ПКМ > Вставить)
4. Нажмите "Получить"
5. Выберите формат и качество
6. Нажмите "Скачать"
7. По завершении: "Открыть папку"

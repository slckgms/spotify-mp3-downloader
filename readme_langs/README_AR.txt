عن البرنامج
-----------
يتيح لك برنامج Spotify MP3 Downloader تحميل الموسيقى مجاناً بجودة عالية
باستخدام روابط Spotify للقوائم أو الألبومات أو الأغاني الفردية.
يتم التحميل عبر YouTube Music.

المميزات
--------
✓ دعم قوائم التشغيل والألبومات والأغاني الفردية
✓ الصيغ: MP3، M4A، OGG، OPUS، AAC، FLAC، WAV، AIFF، ALAC
✓ الجودة: 128 / 192 / 256 / 320 kbps
✓ بيانات وصفية تلقائية (الاسم، الفنان، الألبوم)
✓ تضمين صورة الغلاف (MP3 و M4A)
✓ إعادة المحاولة تلقائياً عند انقطاع الاتصال
✓ دعم قوائم التشغيل التي تحتوي على أكثر من 100 أغنية
✓ دعم 12 لغة

طريقة الاستخدام
---------------
1. افتح قائمة/ألبوم/أغنية على Spotify
2. مشاركة > نسخ الرابط
3. الصق الرابط (Ctrl+V أو انقر بزر الماوس الأيمن > لصق)
4. انقر على "جلب"
5. اختر الصيغة والجودة
6. انقر على "تنزيل"
7. عند الانتهاء: "فتح المجلد"

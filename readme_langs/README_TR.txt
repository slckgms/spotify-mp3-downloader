PROGRAM HAKKINDA
----------------
Spotify MP3 İndirici, Spotify playlist, albüm veya tek şarkı linklerini
kullanarak müzikleri ücretsiz ve yüksek kalitede indirmenizi sağlar.
İndirme işlemi YouTube Music üzerinden gerçekleştirilir.

ÖZELLİKLER
-----------
✓ Playlist, albüm ve tek şarkı desteği
✓ MP3, M4A, OGG, OPUS, AAC, FLAC, WAV, AIFF, ALAC format desteği
✓ 128 / 192 / 256 / 320 kbps kalite seçenekleri
✓ Otomatik metadata (şarkı adı, sanatçı, albüm)
✓ Kapak resmi gömme (MP3 ve M4A)
✓ İnternet kesilince otomatik bekleme ve devam
✓ 100'den fazla şarkılı playlist desteği
✓ 12 dil desteği

KULLANIM
--------
1. Spotify'da istediğiniz playlist/albüm/şarkıyı açın
2. Paylaş > Linki Kopyala ile linki kopyalayın
3. Programda linki yapıştırın (Ctrl+V veya sağ tık > Yapıştır)
4. "Getir" butonuna basın — şarkı listesi yüklenecektir
5. Format ve kalite seçin
6. "İndir" butonuna basın
7. İndirme tamamlandığında "Klasörü Aç" ile müziklere ulaşın

DESTEKLENEN SPOTIFY LİNK TİPLERİ
----------------------------------
• https://open.spotify.com/playlist/...
• https://open.spotify.com/album/...
• https://open.spotify.com/track/...

NOTLAR
------
• Uygulama internet bağlantısı gerektirir
• İlk indirme biraz uzun sürebilir (YouTube araması yapılır)
• 100'den fazla şarkılı playlistlerde şarkı detayları alınırken
  kısa bir bekleme süresi olabilir

ÜBER DAS PROGRAMM
-----------------
Der Spotify MP3 Downloader ermöglicht das kostenlose Herunterladen von Musik
in hoher Qualität über Spotify-Playlist-, Album- oder Titellinks.
Downloads werden über YouTube Music durchgeführt.

FUNKTIONEN
----------
✓ Unterstützung für Playlists, Alben und einzelne Titel
✓ Formate: MP3, M4A, OGG, OPUS, AAC, FLAC, WAV, AIFF, ALAC
✓ Qualitätsoptionen: 128 / 192 / 256 / 320 kbps
✓ Automatische Metadaten (Titel, Künstler, Album)
✓ Cover-Einbettung (MP3 und M4A)
✓ Automatischer Neuversuch bei Verbindungsabbruch
✓ Playlists mit mehr als 100 Titeln unterstützt
✓ 12 Sprachunterstützung

VERWENDUNG
----------
1. Öffnen Sie die gewünschte Playlist/Album/Titel auf Spotify
2. Teilen > Link kopieren
3. Link einfügen (Strg+V oder Rechtsklick > Einfügen)
4. Auf "Abrufen" klicken
5. Format und Qualität wählen
6. Auf "Herunterladen" klicken
7. Nach Abschluss: "Ordner öffnen"

프로그램 소개
-------------
Spotify MP3 다운로더는 Spotify 플레이리스트, 앨범 또는 노래 링크를 사용하여
무료로 고품질 음악을 다운로드할 수 있습니다.
다운로드는 YouTube Music을 통해 수행됩니다.

기능
----
✓ 플레이리스트, 앨범, 단일 노래 지원
✓ 지원 형식: MP3, M4A, OGG, OPUS, AAC, FLAC, WAV, AIFF, ALAC
✓ 음질 옵션: 128 / 192 / 256 / 320 kbps
✓ 자동 메타데이터 (제목, 아티스트, 앨범)
✓ 커버 아트 삽입 (MP3 및 M4A)
✓ 인터넷 끊김 시 자동 재시도
✓ 100곡 이상의 플레이리스트 지원
✓ 12개 언어 지원

사용 방법
---------
1. Spotify에서 원하는 플레이리스트/앨범/노래를 열기
2. 공유 > 링크 복사
3. 링크 붙여넣기 (Ctrl+V 또는 우클릭 > 붙여넣기)
4. "가져오기" 클릭
5. 형식과 음질 선택
6. "다운로드" 클릭
7. 완료 후: "폴더 열기"
